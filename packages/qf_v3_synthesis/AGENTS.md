# AGENTS.md - Synthesis Package

This package is read-only with respect to labs, benchmarks, protocol schemas, and currentness docs.

Synthesis may:

* discover exports,
* validate exports,
* index fixture or future run records,
* summarize records,
* write generated outputs under `generated/`.

Synthesis must not:

* mutate lab files,
* mutate benchmark files,
* mutate protocol schemas,
* mutate currentness docs,
* declare winners,
* graduate methods,
* claim fixture records are evidence,
* or become portfolio authority.

Generated outputs are non-authoritative and ignored by default.

## No Hidden Ranking

Synthesis must not imply a winner through ordering, labels, score summaries, or generated prose unless there is an explicit comparative evaluation path.

If records are sorted for deterministic output, say what mechanical sort key is used.

If synthesis reports metrics, it must describe them as inspection aids, not method authority, graduation evidence, or portfolio decisions.
