# Generated Outputs

Generated synthesis outputs are reproducible, ignored by default, and non-authoritative.

`qf-v3-synthesis` may write files such as:

- `generated/synthesis_summary.json`
- `generated/synthesis_summary.md`

Generated outputs must not mutate labs, benchmark manifests, protocol schemas, currentness docs, registry docs, or lab records. They do not declare winning methods, graduate methods, or become portfolio authority.
