---

## Git Boundary

Allowed in git:

* protocol-valid proposed MethodCard and ExperimentCard planning records,
* admission checklist,
* run admission update,
* metadata-safe prompt template with placeholders only,
* evaluator plan,
* source/privacy boundary,
* prompt/config recording plan,
* stop condition.

Not allowed in git:

* raw source text,
* private notes,
* provider payloads,
* raw model traces,
* prompt traces containing source text,
* secrets,
* generated synthesis summaries.

---

## Local-Only Materials

Source text must remain in a local or ignored path until a later explicit policy allows committing it.

This planning packet does not store the raw source material.
